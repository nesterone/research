# Voice UI Builder

Прототип приложения с голосовым управлением для визуального построения интерфейсов в реальном времени.

## Концепция

Voice UI Builder позволяет создавать UI компоненты с помощью голосовых команд. Компоненты появляются на экране с "жидкой" анимацией, проходя через две фазы:

1. **Wireframe** (1.5 сек) - полупрозрачный контур с пунктирной границей
2. **Styled** - полностью стилизованный компонент

Это создает эффект материализации интерфейса прямо на ваших глазах.

## Возможности

### Голосовое управление
- Распознавание голоса на русском языке
- Непрерывное прослушивание команд
- Отображение транскрипта в реальном времени

### Поддерживаемые команды

| Команда | Результат |
|---------|-----------|
| "мне нужен редактор" | Создается текстовый редактор |
| "сделай мне логин страницу" | Создается форма входа |
| "нужна кнопка" | Создается кнопка |
| "создай карточку" | Создается карточка с контентом |
| "очисти" / "удали все" | Очистка всех компонентов |

### Компоненты

**1. Редактор (Editor)**
- Текстовая область для ввода
- Моноширинный шрифт
- Автоматическое изменение размера

**2. Логин страница (Login)**
- Поля для email и пароля
- Кнопка входа
- Полностью стилизованная форма

**3. Кнопка (Button)**
- Градиентный фон
- Hover эффекты
- Анимация нажатия

**4. Карточка (Card)**
- Заголовок и текст
- Белый фон с тенью
- Адаптивная ширина

## Анимации

### Появление компонента
- **Duration**: 0.6s
- **Easing**: cubic-bezier(0.68, -0.55, 0.265, 1.55)
- **Effect**: Масштабирование от 0.5 с движением вверх

### Материализация (Wireframe → Styled)
- **Duration**: 0.8s
- **Stages**:
  - Wireframe: полупрозрачность, пунктирная граница
  - Transition: плавное появление цветов
  - Styled: полная непрозрачность, все стили

### Индикатор прослушивания
- **Animation**: pulse (1.5s infinite)
- **Effect**: пульсирующее масштабирование микрофона

## Технический стек

- **Frontend**: React 18 (CDN)
- **Transpiler**: Babel Standalone
- **API**: Web Speech API (webkitSpeechRecognition)
- **Styling**: CSS3 с анимациями и transitions
- **Layout**: Absolute positioning с динамическим каскадом

```javascript

recognition.continuous = true;
recognition.interimResults = true;
recognition.lang = 'ru-RU';
```

## Использование

### Запуск
1. Откройте `index.html` в браузере Chrome или Edge
2. Разрешите доступ к микрофону
3. Нажмите на кнопку микрофона
4. Начните говорить команды

### Примеры команд
```
"Мне нужен редактор"
"Сделай мне логин страницу"
"Создай кнопку"
"Добавь карточку"
"Очисти всё"
```

### Управление компонентами
- **Удалить**: наведите на компонент и нажмите × в правом верхнем углу
- **Очистить всё**: нажмите кнопку "Очистить" на панели управления

## Ограничения

1. **Браузеры**: Web Speech API работает только в Chrome/Edge
2. **Язык**: Настроен на русский (можно расширить)
3. **Позиционирование**: Базовая логика каскада (без drag & drop)
4. **Персистентность**: Компоненты не сохраняются при перезагрузке
5. **Компоненты**: Ограниченная библиотека (4 типа)
